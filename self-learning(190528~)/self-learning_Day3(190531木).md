### 25강 패킹 언패킹

패킹 : 여러 개의 변수를 한 개의 변수로 묶기.
언패킹 : 한 개의 변수에 저장된 값을 여러 개의 변수에 풀어 뿌리기.

```python
my_tuple = 1,2,3 # Packing
num1, num2, num3 = my_tuple # Unpacking
num1, num2 = num2, num1 # Packing, Unpacking을 응용한 예.
```
